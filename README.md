# Enhanced Quick Sort Visualizer

This project visualizes the Quick Sort algorithm step-by-step with a clean, interactive interface.

## Features
- Real-time bar chart visualization
- Pivot, comparison, and swap highlighting
- Next & Previous frame navigation
- **Autoplay mode**
- Custom or random array input
- Color-coded legend
- Fully commented code
- Mermaid flowchart included

## Run Instructions
```
pip install -r requirements.txt
python app.py
```

## Files
- `app.py` — main application
- `README.md`
- `requirements.txt`
- `flowchart.mmd` — Mermaid diagram
